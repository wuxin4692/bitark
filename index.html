Universal distributed storage network protocol

Our Vision

We are committed to creating a rich data storage application ecology for human beings, building a common data storage platform, overcoming the obstacles between different data storage platforms, allowing data to flow and use freely and at low cost among various storage platforms, and realizing the real permanence of human data.

Solution

Improved development efficiency

Developers only need to select the corresponding components to develop in BitArk development environment according to their own needs. BitArk protocol provides API interface to adapt to various storage platforms.

Reduce capital expenditures and optimizes costs

We provide access to multiple storage platforms for developers. Developers can easily use the resources of various storage platforms, without building their own storage nodes.

Convenient customized development

Developers can customize data storage products and solutions according to customer needs, use blockchain technology to realize completely decentralized distributed storage.

More friendly to developers

Under the framework of BitArk protocol, the distributed storage application ecology of IPFs will be gradually enriched.

Increases scale and performance of the workloads

The storage infrastructure BitArk interface with keeps expanding. Once a better solution is identified, the developer will have option for better scalability.

cross platform cooperation protocol

BAP (BitArk Protocol) provides storage technology and storage space. It establishes a common storage policy across solutions. BAP supports the self selection and intelligent offering among the different distributed storage solutions for the user.
